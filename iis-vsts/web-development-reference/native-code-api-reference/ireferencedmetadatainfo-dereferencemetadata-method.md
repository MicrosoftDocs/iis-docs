# IReferencedMetadataInfo::DereferenceMetadata Method
Releases the referenced context metadata.  
  
## Syntax  
  
```cpp  
virtual VOID DereferenceMetadata(  
   VOID  
) = 0;  
```  
  
### Parameters  
 This method accepts no parameters.  
  
## Return Value  
 This method returns no values.
