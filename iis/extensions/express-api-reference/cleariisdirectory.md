# ClearIISDirectory
Deletes the IIS Express directory for the current user.  
  
## Syntax  
  
```cpp  
HRESULT  
ClearIISDirectory();  
  
```  
  
## Return Value  
 `HRESULT`  
  
## Remarks  
 This function deletes all related content, including configuration files and log files.
