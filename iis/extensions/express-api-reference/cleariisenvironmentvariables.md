# ClearIISEnvironmentVariables
Removes any dynamic environment variables previously injected using a call to [ApplyIISEnvironmentVariables](../../extensions/express-api-reference/applyiisenvironmentvariables.md).  
  
## Syntax  
  
```cpp  
    HRESULT  
    ClearIISEnvironmentVariables(  
    );  
  
```  
  
## Return Value  
 `HRESULT`
