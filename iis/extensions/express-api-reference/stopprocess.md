# StopProcess
Stops the IIS Express process for the specified process ID.  
  
## Syntax  
  
```cpp  
    HRESULT  
    StopProcess(   
        [in] DWORD dwPid  
    );  
  
```  
  
### Parameters  
  
- [in] `dwPid`  
  
     The ID of the process to stop.  
  
## Return Value  
 `HRESULT`
