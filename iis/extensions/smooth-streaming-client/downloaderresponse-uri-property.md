# DownloaderResponse.Uri Property

**Applies to:** Windows Store apps only

Gets the uri for the response.

## Syntax

```csharp
public Uri Uri { get; }
```

```cpp
public:
virtual property Uri^ Uri {
Uri^ get () sealed;
}
```

```jscript
final function get Uri () : Uri
```

## Property Value

Type: Uri. The URI, which may be original or modified.
