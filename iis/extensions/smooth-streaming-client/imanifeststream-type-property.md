# IManifestStream.Type Property

**Applies to:** Windows Store apps only

Gets the stream 'Type' field.

## Syntax

```csharp
MediaStreamType Type { get; }
```

```cpp
property MediaStreamType Type {
MediaStreamType get ();
}
```

```jscript
function get Type () : MediaStreamType
```

## Property Value

The stream 'Type' field.
