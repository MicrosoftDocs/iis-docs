# IManifestTrack Interface.Bitrate Property

**Applies to:** Windows Store apps only

Gets the track 'Bitrate' field.

## Syntax

```csharp
uint Bitrate { get; }
```

```cpp
property unsigned int Bitrate {
unsigned int get ();
}
```

```jscript
function get Bitrate () : uint
```

## Property Value

The track 'Bitrate' field.
