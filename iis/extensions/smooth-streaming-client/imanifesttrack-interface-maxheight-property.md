# IManifestTrack Interface.MaxHeight Property

**Applies to:** Windows Store apps only

Gets the track 'MaxHeight' field.

## Syntax

``` csharp
uint MaxWidth { get; }
```

``` c++
property unsigned int MaxWidth {
unsigned int get ();
}
```

``` jscript
function get MaxWidth () : uint
```

## Property Value

The maximum height of the track.
