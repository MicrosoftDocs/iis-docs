# RequestUri Property

**Applies to:** Windows Store apps only

Gets the original request URI.

## Syntax

```csharp
public Uri RequestUri { get; }
```

```cpp
public:
virtual property Uri^ RequestUri {
Uri^ get () sealed;
}
```

```jscript
final function get RequestUri () : Uri
```

## Property Value

The request URI.
