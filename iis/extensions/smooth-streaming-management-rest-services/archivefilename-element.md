# ArchiveFileName Element

Specifies the archive file name that is associated with the input stream.

```xml
<ArchiveFileName>
  file-name
</ArchiveFileName>
```

### Child Elements

None.

### Parent Element

[InputStream Element](inputstream-element.md)
