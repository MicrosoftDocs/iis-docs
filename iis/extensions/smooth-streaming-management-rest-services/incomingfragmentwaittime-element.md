# IncomingFragmentWaitTime Element

Specifies the amount of time that has passed since the last fragment was received.

```xml
<IncomingFragmentWaitTime>
  "wait-time"
</IncomingFragmentWaitTime>
```

### Child Elements

None.

### Parent Element

[Track Element](track-element.md)
