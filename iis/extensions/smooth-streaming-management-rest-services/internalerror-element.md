# InternalError Element

If present, specifies the HResult code of the internal error that prompted the error represented by the parent Error element.

```xml
<ErrorCode>
  "error-code"
</ErrorCode>
```

### Child Elements

None.

### Parent Element

[Error Element](error-element.md)
