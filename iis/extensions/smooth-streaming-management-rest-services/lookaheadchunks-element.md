# LookAheadChunks Element

Specifies the number of lookahead fragments that the server buffers so that requests for the subsequent buffered fragments in a stream can be optimized.

    <LookAheadChunks>
      "look-ahead-chunks"
    </LookAheadChunks>

## Child Elements

None.
