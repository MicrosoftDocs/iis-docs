# Name Element

Specifies the track name created by the encoder.

```xml
<Name>
  "name"
</Name>
```

### Child Elements

None.

### Parent Element

[Track Element](track-element.md)

### Remarks

The track name often indicates the track type, such as audio or video, or a custom track type such as a textstream type.
