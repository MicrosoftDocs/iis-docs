# Property Element

Defines a property that is associated with a stream source or destination.

    <Properties>
      <Property name="propertyname">
        "propertyvalue"
      </Property>
    </Properties>

### Attribute

|Attribute|Description|
|--- |--- |
|name|Required. The name of the property.|

### Child Elements

None.

[Properties Element](properties-element.md)
