# State Element (InputStream)

Specifies the state of the input stream (Idle, Started, or Stopped).

```xml
<State>
  "Idle|Started|Stopped"
</State>
```

### Child Elements

None.

### Parent Element

[InputStream Element](inputstream-element.md)
