# Tracks Element

Provides information about the tracks within the live streams that are received by the publishing point.

```xml
<Tracks>
  <Track>
    <Name />
    <EncodedBitRate />
    <IncomingBitRate />
    <IncomingFragmentWaitTime />
    <IncomingFragmentTimeStamp />
    <IncomingFragmentDuration />
    <RequestRate />
  </Track>
</Tracks>
```
