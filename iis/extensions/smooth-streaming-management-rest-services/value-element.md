# Value Element

Specifies a publishing point state (Idle, Starting, Started, Stopping, Stopped, ShuttingDown, or Error).

    <Value>
      "Idle|Starting|Started|Stopping|Stopped|ShuttingDown|Error"
    </Value>

## Child Elements

None.
