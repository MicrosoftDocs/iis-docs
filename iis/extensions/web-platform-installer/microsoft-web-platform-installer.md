# Microsoft Web Platform Installer

The Microsoft Web Platform Installer (WebÂ PI) is a free tool that makes it simple to download, install and keep up-to-date with the latest components of the Microsoft Web Platform, including Internet Information Services (IIS), IIS Media Platform technologies, SQL Server Express, .NET Framework, and Visual Web Developer.

## In This Section

The Web Platform Installer includes a set of Application Programming Interfaces (APIs) that provide programmatic access to the Web Platform Installer user interface, packages, installers, and associated feeds, group, and other objects.
