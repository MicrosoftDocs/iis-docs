# DestroyCompression Function
Destroys the specified compression context.  
  
## Syntax  
  
```cpp  
VOID WINAPI DestroyCompression(  
   IN PVOID context  
);  
```  
  
### Parameters  
 `context`  
 [OUT] Pointer to the compression context to destroy.  
  
## Return Value  
 This function does not return a value.
