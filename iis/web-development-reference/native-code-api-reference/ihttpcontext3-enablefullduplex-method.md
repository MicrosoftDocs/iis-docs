# IHttpContext3::EnableFullDuplex Method
Enables an asynchronous full-duplex connection for this context.  
  
## Syntax  
  
```cpp  
virtual VOID EnableFullDuplex(  
   VOID  
) = 0;  
```  
  
### Parameters  
 This method accepts no parameters.  
  
## Return Value  
 This method does not return a value.
