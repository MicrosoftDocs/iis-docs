# IWebSocketContext::CancelOutstandingIO Method
Cancels all remaining input or output operations.  
  
## Syntax  
  
```cpp  
virtual VOID CancelOutstandingIO(  
   VOID  
) = 0;  
```  
  
### Parameters  
 This method takes no parameters.  
  
## Return Value  
 This method does not return a value.
