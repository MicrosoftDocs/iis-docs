# IHttpResponse::CloseConnection Method
Closes he connection and sends a reset packet to the client.  
  
## Syntax  
  
```cpp  
virtual VOID CloseConnection(  
   VOID  
) = 0;  
```  
  
#### Parameters  
 This method accepts no parameters.  
  
## Return Value  
 This method does not return a value.
